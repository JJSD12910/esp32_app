# ESP32 触控答题演示系统用户操作手册

## 第 1 章 软件概述
- 软件名称：ESP32 触控答题演示系统（基于项目代码实际实现）
- 运行平台：ESP32-S3，配备 3.49 英寸电容触控 LCD
- 主要用途：登录后从服务器获取题库，完成选择题测试并查看成绩
- 面向对象：需要进行现场答题演示或学习测验的用户

## 第 2 章 运行环境与启动方式
- 硬件环境：ESP32-S3 主控、3.49 英寸触控屏、Wi-Fi 网络、机身实体按键（电源键、Boot 键）
- 软件运行条件：设备已配置好 Wi-Fi（用于登录、下载题目与上传成绩），保持供电稳定
- 启动方式：上电即自动启动，系统完成屏幕、触控和网络初始化后进入登录界面
- 登录准备：确保设备连接到预设 Wi-Fi 覆盖范围内，再进行账号与密码输入

【图片说明占位：设备上电后初始显示界面截图】

## 第 3 章 界面与功能总体说明
- 界面形式：基于 LVGL 的图形化界面，可触控、可滚动
- 登录界面：左侧账号/密码输入框，中央数字滚轮选择 0~9，右侧“OK”“DEL”“Sign In”按钮，底部状态提示
- 主功能界面（主页）：提供“Download”“Start Test”“View Results”三个按钮，用于下载题目、开始测试、查看成绩
- 测试界面：展示题目、当前进度、选项文字，以及底部 A/B/C/D 四个大按钮和“Submit”提交键
- 结果界面：显示总分、正确率、正确/错误数、错误题清单，以及“Retry”“Back”按钮

## 第 4 章 答题功能操作说明（重点）
- 界面组成：
  - 顶部：题目文本与当前进度（如 1/10）
  - 中部：四个选项的文字区域，可上下滚动查看完整内容
  - 底部按钮区：A/B/C/D 四个可选按钮（选中后变色高亮），以及“Submit”提交键
- 操作步骤：
  1. 在主页点击“Download”，设备通过 Wi-Fi 下载题目集，下载失败会弹出提示，不会进入测试界面。
  2. 下载成功后点击“Start Test”，进入测试界面并自动显示第 1 题。
  3. 阅读题干与选项，点击底部的 A/B/C/D 按钮选择答案；被选按钮及对应选项文字会高亮。
  4. 点击“Submit”提交当前题目：若未选择选项，会提示“Select an option”；提交后自动跳到下一题。
  5. 最后一题提交后，系统计算得分并尝试上传成绩。
- 补充说明：每题加载时界面自动回到顶部，进度标签实时显示当前题号/总题数。

## 第 5 章 测试流程与结果展示
- 完成全部题目后，系统统计本地得分，并以当前用户标识和试卷标识将成绩上报服务器；成功会弹出“Results uploaded...”提示并返回主页，失败则提示上传失败。
- 在主页点击“View Results”可查看最近一次测试结果；未进行测试时会提示“ No test taken yet”。
- 结果界面内容：
  - 概览：Score（得分/总题数）、Acc（正确率 %）、Time（预留占位）
  - 数量：Correct（正确数）、Wrong（错误数）
  - 错题列表：按题号列出“Your/Ans”对照，仅供查看，不可点击
  - 操作：Retry（重新开始当前题库测试）、Back（返回主页）

## 第 6 章 按键与交互说明
- 触控操作：登录、下载、答题、提交、查看结果等均通过触摸屏完成。
- 实体按键：
  - 电源键（GPIO16）：短按与单击会被监测，长按会在电池供电状态下通过 IO 扩展器发送关断信号（VBAT 关闭）；松开后内部电源标志复位，后续可再上电使用。
  - Boot 键（GPIO0）：当前代码中仅进行事件上报，日常使用不需操作。
- 背光：上电后默认点亮并保持，正常模式下不会自动调节。

## 第 7 章 软件运行流程总结
1. 上电启动，点亮背光并初始化屏幕/触控/Wi-Fi
2. 显示登录界面，按数字滚轮和 OK/DEL 输入账号与密码，点击“Sign In”
3. 登录成功后进入主页；如需测试先点击“Download”获取题目
4. 点击“Start Test”进入答题界面，逐题选择 A/B/C/D 后点击“Submit”提交并跳转下一题
5. 最后一题提交后系统计算成绩并上报服务器
6. 返回主页后可点击“View Results”查看成绩，或点击“Retry”在结果界面重新开始测试
7. 需要关机时长按电源键执行电池侧的关断操作
