# UI 英文与中文对应整理

## 说明
- 本次已替换按钮、状态提示、toast、结果页等 UI 英文文案。
- 答题正文显示区域保留原样，不改 `Loading...`、`Q%d`、`A/B/C/D`。
- 中文 UI 统一使用 `font_24_cn.c` 对应字体。

## 本次实际替换对照
- Account or password empty -> 账号或密码为空
- Signing in... -> 正在登录...
- Network request failed -> 网络请求失败
- Login succeeded but token missing -> 登录成功，但凭证缺失
- Login successful -> 登录成功
- Invalid account or password -> 账号或密码错误
- Login failed -> 登录失败
- Next -> 下一题
- Finish -> 完成
- Session expired, please log in again -> 会话已过期，请重新登录
- No access to this exam -> 无权访问该考试
- No available exam / exam not found -> 无可用考试或考试不存在
- Exam state conflict or already submitted -> 考试状态冲突或已提交
- Download failed -> 下载失败
- Invalid data -> 数据无效
- Out of memory -> 内存不足
- Saved after retry -> 重试后已保存
- Retrying... -> 正在重试...
- No permission to submit answer -> 无权提交答案
- Attempt or question not found -> 作答或题目不存在
- Attempt closed or already submitted -> 作答已关闭或已提交
- Save failed, try again -> 保存失败，请重试
- Please download questions first -> 请先下载题目
- Attempt id missing, please download again -> 作答编号缺失，请重新下载
- Completed -> 已完成
- No permission to submit -> 无权提交
- Attempt not found -> 作答不存在
- Answers saved, final submit failed -> 已保存作答，但最终交卷失败
- Select an option -> 请选择一个选项
